# Witch Bind Extended
This mod makes it possible to customize the size of the 
Witch Crest's tentacle bind.

By default, the tentacles are made 20% larger, but this can be
adjusted using the Bepinex menu (F1). There is an 
additional setting for how large the tentacles should be
if Longclaw is equipped.
